# Another Circle of Fifths

A simple, fast, beautiful circle of fifths — explore keys, modes, chords and how
they relate, with Camelot and solfège notation, an ear-training quiz, sound, and
shareable links.

**▶ Live app: https://dafaspace.github.io/fifths/**

Works offline and installs to your home screen — open the link and choose
"Add to Home Screen".

## Features
- Interactive circle of fifths (major + minor rings), light & dark themes
- Tonal / Modal views, diatonic chords, key relationships
- Camelot wheel & solfège notation, transpose, playback
- Ear-training quiz
